[[modules-indices]]
== 索引

索引模块控索引相关的设置，这些设置全局地管理所有索引，而不是在每个索引级别配置。

可用设置包括：

<<circuit-breaker,断路器>>::

    断路器设置内存使用限制，以避免内存不足异常。

<<modules-fielddata,字段数据缓存>>::

    设置内存中字段数据（fielddata）缓存使用的堆大小限制。

<<query-cache,节点查询缓存>>::

    配置用于缓存查询结果的堆大小。

<<indexing-buffer,索引缓冲区>>::

    控制分配给索引过程的缓冲区的大小。

<<shard-request-cache,分片请求缓存>>::

    控制分片级请求缓存的行为。

<<recovery,恢复>>::

    控制分片恢复过程的资源限制。
